# GitHub Pages Upload

Upload the contents of this folder to the root of a GitHub repository.

Files:
- `index.html` - main interactive site
- `workbook.html` - print-friendly workbook
- `.nojekyll` - tells GitHub Pages to publish files directly

After enabling GitHub Pages for the repository, the site entry page will be:
- `index.html`

The printable workbook opens from the button inside the main site and is also directly available as:
- `workbook.html`
